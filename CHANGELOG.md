## [Unreleased]

### Added
- Initial implementation of MCS networking provider
- Support for Multi-Cluster Services API
- ServiceExport resource creation
- Cross-cluster DNS resolution via MCS
- Compatible with Submariner, Cilium Cluster Mesh...

### Dependencies
- Strimzi Cluster Operator 0.48.0+
- Kubernetes 1.27+ with MCS API support (Strimzi 0.48.0+ supports this Kube version)


### Added
- Initial plugin structure
- MCS networking provider implementation
- Maven build configuration
- Documentation (README, BUILD, CONTRIBUTING)
